## Legend

**Status Values:**
- **Active**: Automation is currently running
- **Inactive**: Automation is paused or disabled
- **Testing**: Automation is in testing phase

**Type Values:**
- **one-time**: Runs once and completes
- **trigger based automation**: Runs when specific conditions/events occur
- **time based recurring**: Runs on a scheduled time basis
